# Email-editor

Service for editing email online

## Running the app

1. clone

```
git clone https://github.com/Hasket510/email-editor
```

2. go in to project

```
cd Email-editor
```

3. install

```
npm install
```

4. start server

```
npx json-server db.json
```

5. development

```
npm run dev
```
